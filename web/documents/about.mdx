Conky is a system monitor for X. It can display almost anything, either on the
root window or in its own window. In addition to its many built-in objects,
Conky can also display information gathered through scripts and external
programs.

Conky includes more than 250 built-in objects, with support for a wide range of
system statistics including `uname`, uptime, CPU usage, memory usage, disk
usage, `top`-style process statistics, and network monitoring. It also includes
built-in IMAP and POP3 support, integration with several music players such as
MPD, XMMS2, and Audacious, and much more. Conky can present this information as
text or through simple progress bars and graph widgets, using a variety of
fonts and colors.

If you would like to get involved, visit the project on GitHub:
https://github.com/brndnmtthws/conky.
